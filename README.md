# rock-paper-scissors-TOP (JavaScript Console Game)
The Odin Project: Rock Paper Scissors

A Simple Rock Paper Scissors game built using JavaScript that runs in the browser's console. The game is played against a computer for 5 rounds.

## How to Play
- The game will prompt you to enter:
    - Rock
    - Paper
    - Scissors
- The computer will randomly choose its move.
- Each round will show:
    - Play choice
    - Computer Choice
    - Round result
    - Current score

After 5 rounds, the final winner is displayed.
